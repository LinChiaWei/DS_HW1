# DS_HW

# **資料預處理：**
    * 合成兩個資料集：台灣電力公司_過去電力供需資訊.csv與本年每日尖峰備轉容量率.csv
    * 將本年度每日尖峰備轉容量率中的備轉容量單位轉換成MW
    * 刪去兩個資料集重疊的日期資料
    * 訓練的資料取兩個欄位：日期與備轉容量率

# **模型建立：**
    使用facebook開發的時間序列預測模型Prophet
    模型架構如下：
        使用加法模型：y(t)=g(t)+s(t)+h(t)+ε(t)
    g(t)：趨勢的影響
    s(t)：季節性的影響
    h(t)：節日的影響
    ε(t)：誤差
